# CodeRabbit review commands

Use CodeRabbit commands only as automation aids; they do not replace
required human approval.

## Common commands used in this repository

- `@coderabbitai review` — trigger or resume a review pass.
- `@coderabbitai full review` — request a broader follow-up review.
- `@coderabbitai resolve` — mark resolved comment threads after the
  underlying fix lands.
- `@coderabbitai help` — list supported commands.

## Usage notes

- Use commands in PR comments, not commit messages.
- Re-run only after a code or documentation change that materially
  addresses feedback.
- Do not dismiss unresolved human feedback just because CodeRabbit is green.
